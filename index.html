Dasturda qaror qabul qilish

Hayotda biz doimo qarorlar qabul qilamiz: "Agar yomg'ir yog'sa — soyabon olaman, aks holda — olmayman". Pythonda ham xuddi shunday! Shartli operatorlar dasturga turli holatlarda turlicha harakat qilishni o'rgatadi.

Data Analyst sifatida biz doimo shartlarni ishlatamiz: foyda bormi yo'qmi, savdo yaxshimi yomonmi, mijoz faolmi yo'qmi — bularning barchasi shartli tekshiruvlar orqali aniqlanadi.

🚦
if

Agar shart rost bo'lsa, kod bajariladi. "Agar ... bo'lsa" degan ma'noni beradi.

🔀
else

Shart yolg'on bo'lganda bajariladi. "Aks holda" degan ma'noni beradi.

📊
elif

Qo'shimcha shartlarni tekshirish. "Agar bo'lmasa, boshqa shart bo'lsa" degan ma'no.

if operatori Shart tekshirish

Eng oddiy shart tekshiruvi

if operatori shartni tekshiradi. Agar shart rost (True) bo'lsa, ichidagi kod bajariladi. Shart yolg'on (False) bo'lsa — tashlab o'tiladi.

if_example.py 
# Oddiy if misoli
yosh = 20

if yosh >= 18:
    print("Siz voyaga yetgansiz ✓")

Diqqat! if dan keyin ikki nuqta (:) qo'yiladi va keyingi qator 4 ta bo'sh joy (indent) bilan boshlanadi.

if + else Ikki yo'l

Agar bo'lmasa, boshqa narsa qil

else — "aks holda" demak. Agar if sharti yolg'on bo'lsa, else ichidagi kod bajariladi. Bu ikki yo'ldan birini tanlash imkonini beradi.

if_else_example.py 
# Yosh tekshirish
age = int(input("Yosh: "))

if age >= 18:
    print("Kirish mumkin ✓")
else:
    print("Kirish mumkin emas ✗")

else da shart yozilmaydi! U faqat if sharti yolg'on bo'lganda ishlaydi.

if + elif + else Ko'p shartlar

Bir nechta shartni tekshirish

elif = "else if" (aks holda agar). Bu bir nechta shartni ketma-ket tekshirish imkonini beradi. Data Analyst ishi uchun juda muhim — chunki biz ko'p kategoriyalar bilan ishlaymiz!

sales_analysis.py 
# 📊 Savdo tahlili - Data Analyst misoli
sales = int(input("Bugungi savdo (so'm): "))

if sales > 100000:
    print("🎉 Ajoyib kun! Savdo a'lo darajada")
elif sales > 50000:
    print("👍 Yaxshi kun! Normal savdo")
elif sales > 20000:
    print("⚠️ O'rtacha kun")
else:
    print("😔 Yomon kun. Tahlil kerak!")

💰 Foyda/Zarar tahlili Real misol

Biznes uchun muhim hisoblash

profit_loss.py 
# 📈 Korxona foyda/zarar tahlili
daromad = int(input("Daromad: "))
xarajat = int(input("Xarajat: "))

natija = daromad - xarajat

if natija > 0:
    print("📈 FOYDA:", natija, "so'm")
elif natija < 0:
    print("📉 ZARAR:", abs(natija), "so'm")
else:
    print("⚖️ Balans nol")

⚖️ Taqqoslash operatorlari Muhim!

Shartlarda ishlatiladigan belgilar

comparison_operators.py 
# Taqqoslash operatorlari ro'yxati

a == b   # Teng (equal)
a != b   # Teng emas (not equal)
a > b    # Katta (greater than)
a < b    # Kichik (less than)
a >= b   # Katta yoki teng
a <= b   # Kichik yoki teng

# Mantiqiy operatorlar
and      # VA (ikkisi ham rost bo'lishi kerak)
or       # YOKI (bittasi rost bo'lsa yetarli)
not      # INKOR (teskari qiladi)

Diqqat! = — qiymat berish, == — taqqoslash. Bularni adashtirmang!
